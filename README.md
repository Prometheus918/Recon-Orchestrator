# Recon-Orchestrator
This tool helps you prepare a Kali Linux recon setup in one step. It installs and checks common tools like subfinder, amass, httpx, nmap, masscan, rustscan, whatweb, wafw00f,   EyeWitness, and wappalyzer using apt, Go, and npm. It also cretes easy to create and interate resports in json and HTML.
